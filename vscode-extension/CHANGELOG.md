# Change Log

All notable changes to the **zeroH** VS Code extension are documented here.

## [0.2.0] - 2026-06-15

### Added
- Initial Visual Studio Marketplace release.
- Commands: **Remember a Fact**, **Ingest Active File**, **Verify Selection
  Against Memory**, **Recall Memories**, **Answer From Memory (no LLM)**, and
  **Ask (grounded completion via your LLM)**.
- Durable SQLite memory store (per-install global storage by default, or a
  custom `zeroh.databasePath`).
- Bring-your-own-LLM settings for OpenAI-compatible and Ollama endpoints.
